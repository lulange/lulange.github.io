Sudoku Solver

Description
The solver has three phases. The first phase attempts to solve with a simple deductive algorithm, but if that doesn't work it will move on to phase 2 which will attempt to solve by adding a possible number and running the phase 1 alg again. If that fails then it will reset and repeat the process until it has looped through all possible numbers to add. Once phase 2 fails, it will begin phase 3. which will begin an exaustive search based on the results from phase 1. Phase 3 will also test for solutions using the phase 1 alg whenever it is logical to do so.
Wait time
Any puzzle that only requires basic techniques will likely be solved in under a tenth of a second by phase 1. Puzzles that are too difficult for phase 1 may still be solved in under a few tenths of a second by phase 2. If a puzzle gets past phase 2 it is likely not a real puzzle and can take anywhere from a few seconds to several minutes or even longer.
